Controller: items     action: datasetview
GETAccept: *, authd - Returns an HTML page with list of zipfiles in {id} with form to unpack the zip file

  • Parameters-> filename = {name of file to be unpacked, including the path}
    -> Id={(Optional)}. if Id is not supplied, the default dataset id for the new dataset will be {dataset_id}-{file_name}
    Returns:Accept:text/html - 302 to splash page for newly created dataset
    Returns:Accept:* 200/201 on update/creation of dataset; 404 if file not found; 415 if file isn't a zip file; 400 if no filename or if unzipping fails
